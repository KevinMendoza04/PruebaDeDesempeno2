Shattered Riffs — New Life Tour

Official landing page for the **New Life Tour** by **Shattered Riffs**. Built with HTML, CSS and vanilla JavaScript — no frameworks or external dependencies.

---

## Preview

> "Reborn through distortion, driven by every broken note."

The page showcases the tour, concert dates, a photo gallery, the band's philosophy, and a form to reserve backstage experiences.

---

## Sections

| Section | Description |
|---|---|
| **Hero** | Full-viewport cover with parallax background image and main CTA |
| **Concerts** | Table with tour dates, cities and venues with buy buttons |
| **New Life Tour** | Tour story and philosophy with a side image |
| **Gallery** | Live concert photo grid |
| **Backstage** | Reservation form for backstage experiences |

---

## Tech Stack

- Semantic HTML5
- CSS3 (custom properties, CSS Grid, Flexbox, animations)
- Vanilla JavaScript (responsive hamburger menu)
- Google Fonts: Playfair Display + Lato

## Getting Started

1. Clone or download the repository
2. Open `landing.html` in your browser

No installation or local server required.

---

## Responsive Design

The layout adapts to three breakpoints:

- **Desktop** (> 1024px) — full multi-column layout
- **Tablet** (≤ 1024px) — reduced columns, image above text
- **Mobile** (≤ 768px) — hamburger menu, single-column layout
